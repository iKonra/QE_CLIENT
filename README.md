# QE CLIENT

Cliente experimental para **Minecraft** con módulos y utilidades integradas.

⚠️ **Advertencia:**
Este proyecto es experimental y está destinado principalmente a aprendizaje y desarrollo.
El uso de modificaciones en servidores multijugador puede resultar en **baneo permanente** dependiendo de las reglas del servidor.

---
